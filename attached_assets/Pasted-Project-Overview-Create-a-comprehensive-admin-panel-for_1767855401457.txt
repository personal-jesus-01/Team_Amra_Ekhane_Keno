Project Overview
Create a comprehensive admin panel for an AI-powered presentation platform that automates slide creation using OCR, Gen AI, and adaptive learning. The platform has these core features:

Document upload with OCR processing

Content filtering by tone/length

AI-generated presentation-ready slides

Slide quality assessment and improvement suggestions

Speech recognition for presentation skills scoring

Bangla speech recognition support

Bkash payment integration (no Visa required)

Design Requirements
Use the provided Tailwind CSS configuration with dark mode support

Color scheme: Primary custom color #886CDD, secondary dark #222222, secondary light #FFFFFF

Fonts: Title (Nico Moji), Primary (Orbitron), Secondary (League Spartan)

Modern, clean dashboard design with sidebar navigation

Responsive for desktop and tablet

Use shadcn/ui components where appropriate
